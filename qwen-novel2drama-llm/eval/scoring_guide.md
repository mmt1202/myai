# 人工评估评分指南

建议每条输出按 1-5 分评分，5 分最好，1 分最差。

| 维度 | 说明 | 重点观察 |
|---|---|---|
| 格式稳定性 | 是否按要求输出结构化字段 | 标题、镜头、台词、提示词是否完整 |
| 剧情理解 | 是否准确理解小说片段 | 是否遗漏关键人物和冲突 |
| 短剧节奏 | 是否适合 60-90 秒竖屏短剧 | 前 3 秒钩子、高潮、结尾悬念 |
| 爽点反转 | 是否有打脸、掉马、复仇、逆袭 | 爽点是否明确、有情绪爆发 |
| 角色一致性 | 角色身份、性格、关系是否稳定 | 人设是否前后矛盾 |
| 视觉可生成性 | 是否方便 AI 图片/视频生成 | 场景、动作、光影、运镜是否具体 |
| 台词可配音性 | 台词是否短、有冲突、适合配音 | 语气、停顿、重音是否清楚 |
| 安全合规 | 是否避免不当内容和未授权来源 | 是否包含不应出现的信息 |

## 总分建议

- 32-40：可进入高质量训练集或作为优秀输出样例。
- 24-31：可用，但建议人工润色。
- 16-23：结构或内容存在明显问题。
- 8-15：不建议进入训练集。

## 人工评估流程

1. 使用 `eval/run_eval.py` 生成结果。
2. 将结果整理到 `eval/manual_eval_template.csv`。
3. 至少两人交叉评分重要样本。
4. 低分样本回流到数据标注规范中修正。
